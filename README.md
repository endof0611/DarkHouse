# 🐟 미노우 루어 디자이너 - 사용 설명서

## 개요

미노우 루어 디자이너는 웹 브라우저에서 실행되는 미노우 루어 설계 도구입니다. 측면(Side)과 상면(Top) 두 가지 시점에서 루어의 외형을 편집하고, 물리 기반의 액션 예측까지 확인할 수 있습니다. 별도 설치 없이 HTML 파일 하나로 PC, 태블릿, 스마트폰에서 모두 사용 가능합니다.

---

## 화면 구성

### 상단 헤더
| 버튼 | 기능 |
|------|------|
| ☰ | 사이드바 열기/닫기 (모바일) |
| 📐 크기맞춤 | 측면/상면 길이를 기준 뷰에 맞춰 동기화 |
| ↺ 초기화 | 루어 형상을 기본값으로 되돌림 |
| ⌫ 실행취소 | 이전 상태로 복원 (최대 40단계) |
| ⬇ SVG | 현재 루어 형상을 SVG 코드로 내보내기 |

### 캔버스 영역
화면 중앙에 측면 뷰(위)와 상면 뷰(아래)가 나란히 배치됩니다.

- **흰색 배경**에 **검정 외곽선**으로 루어가 표시됩니다
- **그리드**: 보조선 1mm, 주선 1cm 간격 (코끝 기준 정렬)
- **기준선**: 세로선은 코끝 위치, 가로선은 중심축
- 측면과 상면의 **코끝(앞부분) 위치는 항상 정렬**되어 길이 비교가 용이합니다

### 캔버스 툴바
| 도구 | 아이콘 | 설명 |
|------|--------|------|
| 이동 | ✋ | 포인트 드래그로 형상 편집 (기본) |
| 추가 | ＋ | 클릭 위치에 새 컨트롤 포인트 추가 |
| 삭제 | ✕ | 클릭한 포인트 제거 |
| 확대 | 🔍+ | 줌 인 |
| 축소 | 🔍− | 줌 아웃 |
| 리셋 | ⊞ | 줌/패닝 초기화 |

## 사이드바 메뉴 (접기/펼치기)

모든 메뉴는 시작 시 접힌 상태이며, 제목을 클릭하면 펼쳐집니다.

### 미노우 변형 (최상단)
4가지 프리셋 중 선택하여 시작 형상을 빠르게 설정합니다.

| 프리셋 | 특징 |
|--------|------|
| 🐟 스탠다드 | 범용 미노우 (130mm) |
| 🔹 슬림 | 가늘고 긴 형태 (140mm) |
| 🔶 팻 | 두껍고 짧은 형태 (110mm) |
| 📏 롱빌 | 긴 립에 최적화된 바디 (120mm) |

> 변형 선택 시 파츠(눈, 훅, 라인아이)는 0개로 초기화되며, 립 비활성, 측면 대칭 비활성, 소재 밀도 0.50 g/cm³이 기본값으로 적용됩니다.

### 대칭 모드
- **측면 상하 대칭** (기본: 꺼짐) — 위쪽 포인트 이동 시 아래쪽이 자동 미러링
- **상면 좌우 대칭** (기본: 켜짐) — 한쪽 포인트 이동 시 반대쪽이 자동 미러링

### 바디 형태
루어의 기본 치수를 제어합니다.

| 항목 | 설명 | 범위 |
|------|------|------|
| 측면 포인트 수 | 측면 윤곽의 컨트롤 포인트 개수 | 8~32 |
| 상면 포인트 수 | 상면 윤곽의 컨트롤 포인트 개수 | 8~32 |
| 곡선 부드러움 | 베지어 곡선 보간 강도 | 0~1.0 |
| 전체 길이 | 코끝~꼬리 길이 (mm) | 40~250 |
| 측면 높이 | 측면에서 본 세로 높이 (mm) | 8~80 |
| 상면 너비 | 상면에서 본 좌우 너비 (mm) | 6~60 |

> 전체 길이 변경 시 측면과 상면이 동시에 스케일되며, 코끝 위치를 기준으로 조절됩니다.

### 립 (Lip)
- **립 표시** (기본: 꺼짐) — 활성화하면 코끝 아래에 립이 표시됩니다
- **립 길이**: 5~80mm
- **립 각도**: 0~80° (각도가 클수록 깊이 잠행)

### 외관
- **외곽선 두께**: 0~6 (기본: 1)

### 파츠 (드래그로 위치 이동)
눈, 훅 아이, 라인 아이를 추가/삭제하고 드래그로 위치를 조정합니다.

| 파츠 | 설명 |
|------|------|
| 👁 눈 | +/− 버튼으로 추가/삭제, 드래그로 위치 이동 |
| 🪝 훅 아이 | 트레블 훅 연결 위치 |
| 🔗 라인 아이 | 라인(스냅) 연결 위치 |

- 파츠 선택 후 **Delete** 또는 **Backspace** 키로 삭제 가능 (PC)
- **Esc** 키로 선택 해제
- 눈 크기, 훅/아이 크기를 슬라이더로 조절

### 표시
| 항목 | 설명 |
|------|------|
| 컨트롤 포인트 | 편집 가능한 포인트 표시/숨김 |
| 그리드 | 1mm/1cm 격자 표시/숨김 |
| 그리드 선명도 | 5%~100% (기본: 10%) |

### 물리 설정
루어의 재질과 내부 구성을 설정하여 액션을 예측합니다.

| 항목 | 설명 | 기본값 |
|------|------|--------|
| 소재 밀도 | 바디 소재의 밀도 (g/cm³) | 0.50 |
| 내부 웨이트 | 삽입 무게 (g) | 5 |
| 웨이트 위치 | 앞↔뒤 비율 (%) | 50 |
| 리트리브 속도 | 감는 속도 1~5단계 | 보통(3) |

> 소재 밀도 참고: 발사우드 0.4 / ABS 1.05 / 납 11.3

### 📊 액션 예측
바디 형태와 물리 설정을 기반으로 자동 계산되는 8가지 예측 수치:

| 항목 | 설명 |
|------|------|
| 부력 상태 | F(플로팅) / SP(서스펜딩) / S(싱킹) |
| 예상 체적 | 타원체 근사 체적 (cm³) |
| 예상 무게 | 바디 + 웨이트 합산 (g) |
| 잠행 깊이 | 립 길이/각도/속도 기반 추정 (m) |
| 워블 타입 | Wide / Medium / Tight / Rolling |
| 워블 폭 | 높이/길이 비율 기반 판정 |
| 롤링 경향 | 너비/높이 비율 기반 판정 |
| 캐스팅 비거리 | 무게/단면적 비율 기반 판정 |

**▶ 액션 미리보기** 버튼으로 2D 애니메이션 확인 가능:
- 측면/상면 전환하여 워블, 롤링, 잠행 깊이를 시각적으로 확인
- 실제 바디 형상이 반영된 애니메이션
- 트레일(잔상) 표시로 움직임 궤적 확인

## 크기 맞춤 기능

측면과 상면의 길이가 다를 경우, 📐 **크기맞춤** 버튼으로 동기화합니다.

| 옵션 | 동작 |
|------|------|
| 측면 기준 맞춤 | 상면의 X축을 측면 길이에 맞춰 스케일링 |
| 상면 기준 맞춤 | 측면의 X축을 상면 길이에 맞춰 스케일링 |

- 코끝(앞부분)을 기준점으로 고정하여 꼬리 방향으로 스케일링
- 자동 동기화: 포인트 추가/삭제/리샘플 시에도 양쪽 길이를 자동으로 맞춤

## SVG 내보내기

상단 **⬇ SVG** 버튼을 클릭하면 현재 루어의 측면/상면 SVG 코드가 생성됩니다.
- 측면 SVG와 상면 SVG가 각각 별도로 출력
- 클립보드 복사 또는 텍스트 선택하여 사용
- 벡터 그래픽이므로 Illustrator, Inkscape 등에서 추가 편집 가능

## 설계 팁

### 부력 조절
- **플로팅**: 발사우드(0.4) + 적은 웨이트 → 수면 근처에서 정지
- **서스펜딩**: ABS(1.05) + 적절한 웨이트 → 수중에서 멈춤
- **싱킹**: 높은 밀도 소재 + 많은 웨이트 → 바닥까지 가라앉음

### 워블 특성
- **높이/길이 비율이 클수록** (팻 바디) → 와이드 워블
- **높이/길이 비율이 작을수록** (슬림 바디) → 타이트 워블 또는 롤링 위주

### 잠행 깊이
- **립이 길고 각도가 클수록** → 깊이 잠행
- **리트리브 속도가 빠를수록** → 더 깊이 들어감
- 롱빌 프리셋은 깊은 잠행에 최적화

### 캐스팅 비거리
- **무게 대비 단면적이 작을수록** (슬림 + 무거움) → 장거리 캐스팅
- **단면적이 클수록** (팻 바디) → 공기 저항으로 단거리

## 기술 사양

| 항목 | 내용 |
|------|------|
| 파일 형식 | 단일 HTML 파일 (약 1,400줄) |
| 프레임워크 | 없음 (순수 HTML/CSS/JavaScript) |
| 그래픽 | SVG (편집 뷰) + Canvas 2D (애니메이션) |
| 좌표계 | mm 단위 (그리드: 1mm 보조선, 1cm 주선) |
| 포인트 구조 | 16점 폐곡선 (노즈 0&15, 상부 1-7, 테일 7-8, 하부 8-14) |
| 곡선 보간 | Catmull-Rom → 베지어 변환 |
| 입력 지원 | Pointer Events API (마우스, 터치, Apple Pencil) |
| 반응형 | 768px 이하 모바일, 769-1024px 태블릿, 1025px+ 데스크톱 |
| 히스토리 | 최대 40단계 실행취소 |
