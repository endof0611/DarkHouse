## 🎮 게임 소개

**어둠 속의 집**은 4개의 스테이지로 구성된 웹 기반 생존 공포 게임입니다. 플레이어는 탈옥한 흉악범의 위협 속에서 문을 잠그고, 도망치고, 미로를 탈출하고, 옥상에서 구출되기까지의 긴박한 여정을 경험합니다.

모든 스테이지는 **제한 시간** 안에 완료해야 하며, 실패 시 심리적 공포 연출과 함께 사망 처리됩니다.

---

## 📖 스테이지 구성

### 🔒 Stage 1 — 우리 집 방어 (30초)

긴급 뉴스 속보가 울린다. 인근 교도소에서 흉악범이 탈옥했다.

- 8개 방(현관, 거실, 부엌, 침실×3, 화장실, 창고)을 돌아다니며 **8개 출입구**(문·창문)를 잠가야 합니다
- 하단 **미니맵**을 클릭하여 방 이동
- 방 안의 문/창문 오브젝트를 직접 클릭하여 잠금
- 불 끄기, 침대·옷장에 숨기 기능 제공
- 30초 후 흉악범이 침입을 시도합니다

### 🏃 Stage 2 — 옆집으로 도망 (20초)

흉악범이 아직 잡히지 않았다. 냉장고는 텅 비어있고, 편의점에 다녀와야 한다.

1. **스토리 선택**: 편의점에서 돌아오는 길에 흉악범과 마주침
2. **도망 미니게임**: 10초 안에 **50번 클릭**하여 도망
3. **문 두드리기**: 옆집 문을 3초 안에 **5번 클릭**하여 열기
4. **옆집 문 잠그기**: 스테이지 1과 같지만 **20초**, 다른 집 구조·가구 배치

> 옆집은 거실 창문 2개, 다용도실 등 우리 집과 다른 구조입니다.

### 🏚️ Stage 3 — 폐건물 미로 탈출 (30초)

흉악범을 피해 폐건물로 도망쳤다. 반대편 출구를 찾아야 한다.

- **4×3 = 12개 방** 그리드 구조의 미로
- 각 방에 여러 문이 있지만 **정답 문은 1개** (출구로 가는 최단 경로)
- 오답 문 클릭 시 **-2초 패널티** + 빨간 플래시
- 흉악범이 **4초마다 플레이어를 향해 이동**
- 미니맵에 현재 위치(📍), 출구(🚪), 흉악범(🔴) 표시
- 같은 방에서 마주치면 즉시 사망

### 🚁 Stage 4 — 옥상 탈출 (40초)

경찰 헬기가 옥상으로 온다. 5층까지 올라가야 한다.

**1~4층** (층마다 반복):
1. **연타 미니게임**: 화면 클릭하여 계단 올라가기 (층마다 필요 클릭 수 증가)
2. **비밀번호 힌트 획득**: 옥상문 4자리 비밀번호 힌트 1개씩 제공
3. **장애물 회피**: 흉악범이 던지는 장애물을 ⬅/➡ 선택하여 피하기 (2초 제한, 실패 시 -3초)

**5층 옥상 도착**:
4. **비밀번호 입력**: 1~4층에서 모은 힌트로 4자리 코드를 키패드에 입력
5. 성공 시 헬기 탈출 → **흉악범 체포 엔딩!**

| 조작 | 설명 |

## 🎨 특징

- **숨겨진 인터랙션**: 클릭 가능한 오브젝트에 호버 효과 없음 — 직접 눌러봐야 알 수 있음
- **심리적 공포 연출**: 실패 시 피 번짐 효과 → 점프스케어 이미지 → 사망 화면
- **프로시저럴 사운드**: Web Audio API로 생성되는 발자국, 문 흔들림, 공포 드론 효과음
- **반응형 디자인**: PC와 모바일 모두 지원 (터치 최적화)
- **단일 HTML 파일**: 외부 의존성 없이 브라우저에서 바로 실행
